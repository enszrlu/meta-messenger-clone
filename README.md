## Overview

Basic Messenger with Facebook Auth. Logged in user will see their message on right handside and all others on left handside.

### Built with

-   Next.js & React.js
-   [tailwindcss](https://tailwindcss.com/) - CSS Framework
-   redis via Upstash
-   NextAuth.js
